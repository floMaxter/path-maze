Finding for a path from start to finish. 

This function is basic, it includes a pathfinding algorithm in a maze. The input receives the coordinates of a point and an object of the \mbox{\hyperlink{class_maze}{Maze}} class, as well as an output stream to a file. A boolean variable is returned from the function, indicating whether the exit was found or not.
